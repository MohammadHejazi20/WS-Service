Lightweight WebSocket Pub/Sub for realtime apps

ws-service is a minimal TypeScript WebSocket server that lets clients subscribe to topics and publish messages in real-time. Simple API. No fuss.

Simple API

Three actions: subscribe, publish, unsubscribe. That's all you need to know.

Quickstart

Run a local server and connect a client. Requires Node.js. Uses pnpm.

Install & run
pnpm install
pnpm dev  # starts ws://localhost:8080
Connect with websocat
websocat ws://localhost:8080
Minimal browser client

Copy and paste this into your browser console to test the connection

const socket = new WebSocket('ws://localhost:8080');

socket.addEventListener('open', () => {
  socket.send(JSON.stringify({ action: 'subscribe', topic: 'chatroom' }));
  socket.send(JSON.stringify({ action: 'publish', topic: 'chatroom', payload: 'Hello!' }));
});

socket.addEventListener('message', (event) => {
  console.log('Message from server:', event.data);
});

WebSocket API

Send JSON messages over the WebSocket. The server parses action, topic, and optional payload.

Subscribe
Begin receiving messages for a topic 
{
  "action": "subscribe",
  "topic": "chatroom"
}
Publish
Broadcast a message to subscribers 
{
  "action": "publish",
  "topic": "chatroom",
  "payload": "Hello, World!"
}
Unsubscribe
Stop receiving messages for a topic 
{
  "action": "unsubscribe",
  "topic": "chatroom"
}
Message schema
{
  "action": "subscribe" | "publish" | "unsubscribe",
  "topic": string,
  "payload"?: unknown
}

Invalid JSON results in a server error response.

Server behavior
  • Subscribes track client IP and topic membership
  • Publish sends payload to all subscribers of a topic
  • Client close auto-unsubscribes from all topics
  • Console shows a live table of topics, subscribers, and messages

Examples

Node.js client
Server-side WebSocket client using the ws library
import WebSocket from 'ws';

const ws = new WebSocket('ws://localhost:8080');

ws.on('open', () => {
  ws.send(JSON.stringify({ action: 'subscribe', topic: 'news' }));
  ws.send(JSON.stringify({ action: 'publish', topic: 'news', payload: { title: 'Breaking' } }));
});

ws.on('message', (data) => {
  console.log('Received:', data.toString());
});
websocat session
Command-line testing with websocat tool
websocat ws://localhost:8080
{"action":"subscribe","topic":"chatroom"}
{"action":"publish","topic":"chatroom","payload":"Hello from terminal"}
